An end-to-end Machine Learning project built using the Ames Housing dataset.
The dashboard demonstrates:
- Data cleaning and preprocessing
- Feature engineering (HouseAge, TotalSF, TotalBath, etc.)
- Categorical encoding and normalization
- Model training using XGBoost, LightGBM, and CatBoost
- Hyperparameter tuning and model evaluation
- Ensemble prediction and deployment using Streamlit
